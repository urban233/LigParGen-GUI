# Installation for Windows Operating System
This page will present a comprehensive in-depth explanation of the installation procedure.
The LigParGen-GUI installation package is approximately **1.5 GB** in size, taking **a minute to download** depending on your internet speed. 
The entire installation process of both components (WSL2 and LigParGen-GUI) should take around **10 minutes**
(with a **hands-on time** of around **3 minutes**) and may require a **restart of the computer**.

## Prerequisites
You must be running Windows 10 version 2004 and higher (Build 19041 and higher) or Windows 11 to use LigParGen-GUI.

If you have a real-time virus scanner software installed (e.g. Avira, Kaspersky, ...), please disable it temporarily during the installation process.
